# Subscriptions

Manage subscription expiry dates and reminders in one place.

## Features

- **Add subscription**: Website name, URL, expiry date, cost, notes, tags.
- **List & filters**: Filter by expiry (expired, within 7/30 days, etc.) and by tag; card or table view.
- **Sync queue**: Bidirectional sync (pending upserts/deletions), optional Telegram reminders.
- **Expiry alerts**: Configurable check interval; Telegram Bot can send expiry reminders.
