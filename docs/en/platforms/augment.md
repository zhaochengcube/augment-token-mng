# Augment

Manage Augment Code access tokens and accounts.

## Basic flow

1. **Generate auth URL**: Click "Generate Augment auth URL" in the app, copy or open the link.
2. **Authorize in browser**: Complete OAuth and copy the JSON response from the browser.
3. **Get token**: Paste the JSON into the app and click "Get access token".
4. **Save & manage**: Save the token and manage it under "Saved tokens".

## Session import

- **Manual**: Follow the instructions in the help on the home page.
- **Auto**: Click "Auto import", sign in to Augment in the browser, then use the import button on the right side of the window.
