# 邮箱管理

管理 Outlook、GPT 邮箱及 iCloud HME 等邮箱相关功能。

## 功能说明

### iCloud HME

iCloud 隐藏邮箱的生成、列表、停用与清理（需 iCloud+ 账号）。

### Outlook

Outlook 邮箱账户管理：

- **添加账户**：支持手动导入（邮箱----Refresh Token----Client ID）和 OAuth 授权两种方式，支持批量导入。
- **Token 管理**：后台定时自动刷新 Token，支持全量手动刷新和失败重试。
- **状态检查**：批量检查账户在线状态（正常/异常/封禁）。
- **邮件查看**：点击账户可直接查看收件箱邮件（通过 Microsoft Graph API）。

### GPTMail

通过 GPTMail API 生成临时邮箱并收取邮件：

- **生成邮箱**：一键生成随机邮箱地址，支持自定义 API Key。
- **收取邮件**：输入邮箱地址即可查询邮件，自动提取验证码。
- **自动轮询**：开启后自动定时检查新邮件。
- **历史记录**：已生成的邮箱地址自动保存，支持搜索和管理。

从侧边栏进入「邮箱管理」后，选择对应服务使用。
