# Augment 平台

管理 Augment Code 访问令牌与账号。

## 基本流程

1. **生成授权 URL**：在应用内点击「生成 Augment 授权 URL」，复制或打开链接。
2. **浏览器授权**：完成 OAuth 后，从浏览器复制返回的 JSON。
3. **获取令牌**：将 JSON 粘贴到应用中，点击「获取访问令牌」。
4. **保存与管理**：保存 Token，在「已保存 Token」中查看、编辑、删除。

## Session 导入

- **手动导入**：按主页帮助按钮的说明操作。
- **自动导入**：点击「自动导入」，在浏览器中登录 Augment 后，在窗口右侧点击导入 Session。

## 插件

- VSCode 及 Fork：使用 [Augment Code Auto](https://github.com/cubezhao/augment-code-auto)。
- JetBrains：从 [Releases](https://github.com/cubezhao/ai-tools-mng/releases) 下载对应插件（使用可能带来封号风险，请自行权衡）。
